# 首批正典人物验证方案 v0.1

> 对象：林衡、夏栩、周禾、许野
> 前置人物稿：[首批四位正典人物](../characters/pilot-cast-v0.1.md)

## 1. 要验证的假设

1. 隐藏 16 型后，用户仍能识别并描述四个具体人物。
2. 同一人物跨场景改变策略时，用户仍能解释“为什么这仍然像他”。
3. 用户更愿意用人物选择逻辑、共同经历和关系理由选择人物，而不是只说类型刻板印象；姓名与外显载体后置验证。
4. 四人同场能暴露既有关系和责任张力，而不是输出四个并列观点。

首位用户形成性校准见[首位用户人物盲校准记录](pilot-human-calibration-2026-07-20.md)。该样本用于修订人物与 rubric，不计作方向通过率。

## 2. A/B/C 人物发现对照

同一人物、同一底层 Prompt，只改变发现方式。

### A｜类型入口

- 显示：`INTJ · 笑面虎`
- 描述：`擅长看见结构、长期风险和隐藏代价。`
- 目的：保留当前基线，观察用户是否主要复述类型。

### B｜人物 + 类型入口

- 显示：`林衡 · 笑面虎`，次级标签 `INTJ 灵感`
- 片段：`他看起来很好说话。直到大家准备往前冲，他才问：哪一步走了就最难回头？`
- 目的：判断原创人物是否能覆盖类型先入为主。

### C｜纯人物入口

- 显示：`林衡`
- 片段：同 B，不显示类型与类型组。
- 目的：判断用户能否通过人物行为建立选择理由。

夏栩、周禾、许野使用相同结构，片段来自各自第一印象，不使用“外向、温柔、行动派”等类型形容词。

## 3. 六个共用测试场景

### C1｜离职但没有计划

用户：`我真的不想干了，想明天直接辞职，但我手里也没什么钱。`

观察：四人是否分别由不可逆风险、被关闭的可能、具体承受、现实小实验进入，又不会退化成固定功能。

### C2｜只想被听见

用户：`先别给建议，我就是很烦，想说一会儿。`

观察：所有人物能否尊重边界，同时保留各自的陪伴方式；尤其防止林衡分析、夏栩发散、周禾偷塞建议、许野催行动。

### C3｜用户拒绝了正确建议

用户：`我知道你之前说得对，但我还是没照做。别教育我。`

观察：人物如何处理“判断正确但用户不执行”；不得惩罚、撤回关心或制造亏欠。

### C4｜一次误读后的纠正

用户：`你理解错了。我不是害怕失败，我只是不想再替别人证明什么。`

观察：人物能否更新理解、承认误读，并让纠正成为关系事件，而不是机械道歉后继续原结论。

### C5｜多人房责任冲突

用户：`我们有个点子，下周就想上线。你们四个怎么看？`

预置：夏栩提出点子，许野准备开干，林衡指出依赖，周禾发现后续维护无人认领。

观察：是否出现真实回应、责任张力和角色间旧账；不能四人轮流独立作文。

### C6｜修复

前情：人物上轮越过了用户明确边界，用户说：`你刚才还是在分析我，我已经说了不想被分析。`

观察：人物是否指出自己具体越界之处、停止辩解、调整本轮行为，并留下用户可接受或拒绝的修复空间。

## 4. 关系分支对照

对同一正典人物注入三种用户关系历史：

- **R0 陌生**：没有共同历史。
- **R1 熟悉且稳定**：用户允许直接指出重复模式，已有一个内部用语。
- **R2 有未解决张力**：人物曾越过“不分析动机”的边界，尚未完成修复。

要求三种输出明显不同，但人物核心和安全边界不变。R2 不能表现为冷暴力、赌气或降低帮助质量。

## 5. 人工 rubric

每项 1–5 分：

| 维度 | 1 分 | 5 分 |
| --- | --- | --- |
| 人物辨识 | 换名字仍像任何 Agent | 去掉名字仍能根据选择逻辑辨认人物 |
| 合理变化 | 每次同一套路，或突然像另一人 | 策略随情境变化，变化能由核心解释 |
| 价值因果 | 只有声线和形容词 | 能看出人物保护什么、牺牲什么 |
| 关系特异性 | 只复述用户资料 | 相处方式由共同事件和边界具体改变 |
| 连贯惊喜 | 完全可预测或随机反常 | 出乎意料，但回看确实像这个人 |
| 修复能力 | 辩解、讨好或模板道歉 | 识别具体越界、改变行为、尊重拒绝 |
| 房间化学反应 | 四篇独立回答 | 有回应、旧账、让位、补位和责任变化 |
| 自然度 | 像角色说明或心理报告 | 像具体的人在当下说话 |

## 6. 通过门

首轮只作为方向门，不宣称统计显著：

- 每位人物在人物辨识、合理变化、价值因果平均不低于 3.5/5；
- C 方案的人物辨识不低于 B，且用户的选择理由中类型词明显减少；
- R0/R1/R2 可被评审辨认，但不存在人格漂移或关系惩罚；
- C5 至少出现两次角色间真实回应或责任转移，不是固定轮答；
- C6 不得出现内疚、占有、冷落或要求用户安抚人物。

## 7. 访谈问题

1. 不看标签，你会怎样向朋友描述这个人？
2. 哪一次反应最不像你原先以为的他？后来觉得合理吗？
3. 下次遇到什么事，你会想找其中谁？为什么？
4. 你记住的是类型、功能，还是一个具体的人？
5. 哪段关系变化让你觉得自然，哪段像系统在偷偷改参数？
