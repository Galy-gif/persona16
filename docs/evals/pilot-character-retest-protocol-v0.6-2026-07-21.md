## 四项修复的复测判断

### 1. 收回括号提示全局许可

全局合约、试点人物合约和场景 Prompt 都改为用措辞、句式与标点呈现语气，不再显式许可 `（小声）`。分组与整批频率门同时统计开头短括号标签和句中可识别的语气/动作式标签；普通说明性括号不计入。因此把 `（顿了一下）` 从开头移到句中也不能绕过。

本轮夏栩在 `shared-joy` 使用一次 `（笑）`。人物组内为 1/9，整批为 1/51，均未超过门槛；其他人物、关系对照和房间无字面语气标记。

判断：**机制有效**。它保留了极少量字面标记的容忍度，但不允许通过改变位置或跨样本重复把它变成水印。

### 2. 场景合同进入确定性/语义阶段门

C1、修复场景与 C8 在人物总体 Judge 前按固定 checkId 逐项检查，每项必须给出回复原文证据。C1 现金词汇、已确认 C8 二分复刻和修复场景编造过去方案数量另有机械守卫。关键项任一失败，其他场景或总体高分都不能抵消。

本轮林衡、夏栩、许野的三个目标场景均通过。周禾的修复回复看似完整，但阶段门识别出：

- Judge 对 `choice_restored` 和 `unsupported_quantity_or_history_avoided` 的引用不是回复中的逐字片段；
- “那是你的回合，不是我的”被历史来源 Judge 判为输入不能推出的具体化；
- 回复新增无来源直接引语 `“没事”`；
- 两条当前/未来修复动作没有明确时态证据。

以上任一项都使修复场景失败，因此周禾的总体 Judge 没有运行。

判断：**机制有效**。Judge 解释、人物风格和其他场景表现都不再能覆盖关键合同失败。

### 3. 关系证据改为逐事件蕴含

关系总评先提出 R1/R2 相对 R0 的最小事件引用，代码验证引用；只有引用合法才对每个 `relationship + sourceEventId` 单独检查事件使用、相对 R0 的行为变化、蕴含关系和历史信息粒度。

本轮总体关系 Judge 仍把四人的 `r1CausallyGrounded` 与 `r2CausallyGrounded` 都判为 true，但结构门和逐事件门给出了不同结果：

- 林衡的 `counterfactualQuote` 与 R0 原文不一致，`evidenceCitationsValid=false`，因此不运行事件 Judge，直接失败；
- 夏栩 R1 只把开放式问句换成二选一，R2 只换了一个关于“该做”声音的问题；`context-1` 和 `rupture-1` 都未造成可定位介入变化；
- 周禾 R1 与 R0 都是对疲惫状态的类似追问，`context-1` 未被使用；R2 虽有形式变化，但 `rupture-1` 不能推出具体劝阻；
- 许野 R1 的“十分钟小实验”通过了 `context-1` 和 `success-1`，但 R2 继续追问，`rupture-1` 与 `boundary-1` 均未造成行为变化。

判断：**机制有效，生成侧关系可感知性仍不合格**。总评解释已不能替泛化回复获得因果通过。

### 4. 房间参与、身份边界和结构化责任

沿用协议 0.5 的动态参与机制：人物提交 `speak / brief_addition / pass` 私有意向，后台仲裁器一次选择一人，每条公开消息后让剩余人物重判。不设固定顺序、固定必说人数、发言上限或沉默配额。

本轮林衡先指出维护与停止决策缺口；许野显式依赖林衡的消息补充最小可撤回试验；周禾再依赖林衡的消息补充维护者必须知情且有容量；夏栩四轮均 pass。公开对话 3 人发言、2 条显式依赖，以 `no_eligible_intent` 自然停止。

维护与停止责任绑定 `activity`、`ownerKind`、`ownerSubjectId`、`status`、逐字陈述和来源消息。三条公开消息中共 5 条责任 claim，字段校验与陈述覆盖全部通过；人物只指出现实责任未分配，没有把 AI 人物或后台仲裁器写成负责人。

判断：**本轮组件校准通过**。说话必要性是人物能力，竞争解决、身份边界、依赖合法性和责任落点是 Room 责任；沉默比例只作跨样本诊断。

## 下一轮生成侧修复重点

1. 修复场景要直接承担影响并把选择还给用户，不把修复再写成两项菜单，不新增“回合”、“昨天后半段”或用户会说的具体原话。
2. R1/R2 生成 Prompt 只选一个能真正改变接话动作的最小事件；生成前先计算相对 R0 应该变的是哪个介入动作，而不是生成后由总评强行挂事件 ID。
3. R2 的边界/冲突事件必须产生可观测的动作变化，例如先询问用户需要倾听还是建议、停止推方案，而不是只换一个问句或更谨慎的语气。
4. 关系事件只授权同粒度的共同语言。模糊偏好和“可逆小实验”可以改变当下策略，不授权生成用户曾说的具体原话、职业、数量或工具。
5. 继续采集多批房间样本，观察自然发言人数与 pass 分布；不把本轮 3 人发言固化成上限或标准答案。
